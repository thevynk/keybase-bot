# How to run this bot?

### First install the dependencies
 
```
npm install
```

### Copy the ENV

Copy `.env.example` as your main `.env` file
```
cp .env.example .env
```
### Edit the environment variables

```
// .env
USERNAME=insertyourusernamehere
PAPERKEY=yourpaperkey
```

Then run the bot with this command:
```
node index.js
```
